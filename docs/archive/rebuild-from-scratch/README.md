# Rebuild From Scratch

이 폴더는 IIHS 다운로드 파이프라인을 처음부터 다시 세워야 할 때 참고하는 아카이브 절차 문서다.

현재 프로젝트의 주 작업이 데이터베이스 구축이라면, 먼저 아래 문서를 본다.

- `docs/README.md`
- `docs/core/database-build-playbook.md`
- `docs/core/research-database-architecture.md`

이 폴더의 단계 문서는 다음 상황에서만 연다.

- 인증 세션 확보 방식이 바뀌었을 때
- discovery 또는 download 파이프라인을 다시 구현해야 할 때
- 신규 환경에서 수집 기준선을 재구축해야 할 때
